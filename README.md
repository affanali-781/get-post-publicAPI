# API Interaction App

## Description
This is a **React-based application** that allows users to interact with APIs through **GET** and **POST** requests. The app provides a clean interface for selecting API actions, fetching data, and submitting new entries.

**Features:**
- Choice page to select GET or POST API actions.
- GET page: Fetches and displays data from a sample API.
- POST page: Allows adding new data via a form.
- Styled with **Tailwind CSS** for modern, responsive UI.
- Navigation handled with **React Router DOM**.
